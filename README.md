# The Factor Zoo Audit

Does the market still pay for the factors academic finance says it should?

## Overview
This project tests whether five canonical stock-market factors (market, size, value, profitability, momentum) maintain their premiums post-2010 and retain statistical significance after realistic trading costs.

## Key Findings
- H1: 5 of 6 factors show decay (premiums shrink post-2010)
- H2: After costs, all factors except market drop below HLZ significance threshold (t ≥ 3.0)
